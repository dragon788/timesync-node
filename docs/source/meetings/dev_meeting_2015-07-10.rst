        * ``(U1 || U2) && (P1 || P2) && (A1 || A2)``
    * Example query: ``/times?user=pop&user=tschuy&activity=meetings&activity=planning&project=timesync&project=pgd``
        * should return all meetings and planning done for timesync or pgd done by tschuy or pop
